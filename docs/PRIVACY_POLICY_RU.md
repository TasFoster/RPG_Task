# Политика конфиденциальности — RPG Трекер

_Последнее обновление: 2026-07-07_

Приложение **RPG Трекер** (далее «Приложение») — трекер задач, привычек и целей
с элементами геймификации. Мы уважаем вашу приватность. Этот документ описывает,
какие данные обрабатываются и как.

## Какие данные обрабатываются

- **Ваш контент.** Задачи, привычки, цели, шаги, отметки выполнения, начисленный
  опыт/валюта, купленная кастомизация, настройки. По умолчанию всё это хранится
  **только на вашем устройстве** (локальная база данных SQLite).
- **Технические данные.** Приложение не собирает аналитику и не использует
  сторонние рекламные SDK.

## Хранение и передача

- **Локально (по умолчанию).** Без учётной записи данные не покидают устройство.
- **Облачная синхронизация (опционально, при появлении аккаунтов).** Если вы
  войдёте в аккаунт, ваш контент будет синхронизироваться с защищённым бэкендом
  (Supabase) для доступа с нескольких устройств. Данные привязаны к вашему
  идентификатору пользователя и защищены правилами доступа (RLS): другие
  пользователи их не видят. Вход остаётся полностью добровольным — Приложение
  работает офлайн без аккаунта.

## Разрешения

- **Уведомления.** Используются для напоминаний о задачах и привычках в заданное
  вами время. Разрешение запрашивается по необходимости, его можно отозвать в
  настройках системы.
- **Точные будильники / полноэкранные уведомления (Android).** Для срабатывания
  напоминаний в назначенное время, в т.ч. в режиме энергосбережения.
- **Планируемые чувствительные разрешения (в будущих версиях, строго опционально
  и выключены по умолчанию):** доступ к статистике использования, специальные
  возможности (AccessibilityService) и показ поверх других приложений — только
  для функций «жёсткого фокуса»/блокировки отвлечений. Каждое включается вручную
  с отдельным экраном-пояснением и согласием; Приложение полностью функционально
  без них.

## Дети

Приложение не предназначено для детей младше 13 лет и не собирает намеренно их
персональные данные.

## Удаление данных

Локальные данные удаляются вместе с удалением Приложения. При использовании
облачного аккаунта вы сможете запросить удаление данных аккаунта.

## Изменения политики

Мы можем обновлять эту политику. Актуальная версия всегда доступна в репозитории
Приложения и/или в его карточке в магазине.
