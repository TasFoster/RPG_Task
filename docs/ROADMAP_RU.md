## 1. 🔴 Мотивация, цитаты и советы по всему приложению

Главная фича этого документа: в удобных местах показывать советы по тайм-менеджменту,
саморазвитию, мотивации и дисциплине, а также мотивационные фразы и цитаты.

### 1.1. Зафиксированные решения

| Параметр | Решение |
|---|---|
| **Тон/подача** | Смешанный: **реальные цитаты и техники в RPG-обёртке** (польза + атмосфера) |
| **Источник контента** | Комбо: встроенный базовый набор + пользовательские + онлайн-паки позже |
| **Язык** | Только русский (на старте; en — если решим локализовать контент позже) |
| **Где показывать** | «Совет дня» на главной · контекстные микро-подсказки · пуш-уведомления · коллекция «Кодекс героя» |

### 1.2. Открытые вопросы (значения по умолчанию — предложение, подтвердить)

1. **Авторство.** _По умолчанию:_ реальные цитаты — с указанием автора (Марк Аврелий, Сенека,
   Дж. Клир «Атомные привычки», Кэл Ньюпорт, Брайан Трейси и т.п.); «обёртка» (вводная фраза
   наставника/питомца) — во вселенной игры. **Избегать:** религия, политика, спорные личности.
   _Нужно ли добавить/исключить конкретных авторов или направления?_
2. **Объём на старте.** _По умолчанию:_ ~150 единиц контента в первой версии. _Мне сгенерировать
   стартовый набор на русском?_ (могу подготовить как готовый датасет для БД).
3. **Управление.** _По умолчанию:_ в настройках — тумблеры по каждому месту показа (Совет дня,
   контекстные, пуш) + лимит частоты (не чаще N в день) + «не повторять последние M».
4. **Формат.** _По умолчанию:_ два типа — короткая **цитата/фраза** и практический **совет-техника**
   (мини-инструкция в 1–2 предложения). _Нужны ли развёрнутые мини-гайды (карточки на абзац)?_

### 1.3. Типы контента и категории

- **Типы:** `quote` (цитата с автором) · `tip` (практическая техника/совет) · `flavor`
  (короткая RPG-реакция: «Ещё один шаг к славе, искатель!»).
- **Категории:** Тайм-менеджмент · Дисциплина · Мотивация · Фокус · Саморазвитие · Стойкость.
- **Техники для «tip»** (примеры наполнения): Помодоро, тайм-боксинг, «съешь лягушку» (worst-first),
  правило 2 минут, принцип Парето 80/20, матрица Эйзенхауэра, «не разрывай цепь» (Сайнфелд),
  дизайн окружения, приёмы против прокрастинации, ритуалы старта/завершения дня.

### 1.4. Где и когда показывать (детально)

- **«Свиток дня» на главной** — один заметный блок, детерминированно меняется раз в сутки
  (одинаков в течение дня), с анимацией «разворачивания». Кнопки: ⭐ в избранное, 🔗 поделиться.
- **Контекстные микро-подсказки** (умные, по ситуации):
  - завершил задачу/привычку → короткий принцип дисциплины или похвала;
  - открыл Помодоро/фокус → техника концентрации;
  - **сорвал стрик** → цитата про стойкость (падать 7 раз — вставать 8);
  - level-up / убил босса-цель → триумфальная фраза;
  - пустой экран (нет задач) → мягкий пинок начать;
  - привязка к **осям навыков** (совет про здоровье при отметке спорт-привычки и т.д.).
- **Пуш-уведомления** (через готовую систему, только Android/iOS): «утренний девиз» и
  «вечерняя рефлексия»; время настраивается; на web — no-op.
- **Коллекция «Кодекс героя»** — отдельный экран: собранные/открытые цитаты, избранное,
  фильтр по категориям, достижение за полную коллекцию.

### 1.5. Геймификация контента

- **«Кодекс героя»** — открывай новые цитаты за активность; собрал всё → достижение + награда.
- **Паки цитат в магазине** — за золото/кристаллы: «Свод стоиков», «Кодекс продуктивности»,
  «Изречения воина». Завязка на существующую экономику и магазин.
- **Тема недели** — «Неделя дисциплины»/«Неделя фокуса»: подобранные советы + бонусный XP.
- **Поделиться цитатой** — рендер красивой карточки в фирменном стиле (щит, #1a1024) для соцсетей.

### 1.6. Черновик модели данных (Drift)

```
tips            (id, type, category, text, author, source, is_premium, pack_id)
tip_packs       (id, title, price_gold, price_gems, is_owned)   // для магазина
user_tips       (id, text, category, created_at)                // свои цитаты
tip_progress    (tip_id, discovered_at, is_favorite)            // Кодекс героя
settings        (+ поля: tips_daily_enabled, tips_context_enabled,
                 tips_push_morning, tips_push_evening, tips_daily_cap)
```
Контент — офлайн-first (как всё приложение); онлайн-паки подтягиваются в `tips` при появлении Supabase.

### 1.7. Этапы внедрения

- **Этап A (MVP): ✅ реализован** — встроенный каталог (~150), «Совет дня» на главной,
  контекстные подсказки (пустой экран задач → мотивация, Помодоро → фокус), тумблер в настройках.
  Код: `lib/features/tips/`. Пул «Совета дня» — цитаты и советы (без RPG-реакций), выбор детерминирован по дате.
- **Этап B: ✅ реализован** — «Кодекс героя» (сбор цитат как лут за активность, избранное,
  награды за категории и полную коллекцию) + пуш-советы (утро/вечер, настраиваемое время).
  Каталог расширен до ~235. Код: `lib/features/codex/`, `lib/features/tips/`.
- **Этап C (доп.):** паки в магазине + темы недели + свои цитаты + «поделиться карточкой».
- **Этап D (доп.):** онлайн-подгрузка паков (после интеграции Supabase, Фаза 2).

**Также реализовано (этот заход):**
- **Хитмап привычек** (GitHub-стиль, адаптивно ~17–26 недель, интенсивность по числу):
  сводка на экране привычек + детальный экран привычки. Код: `lib/features/habits/presentation/`.
- **Трекер сна и циркадные ритмы**: ручной ввод/правка, калькулятор циклов (90 мин, обе стороны),
  опросник хронотипа, журнал + график длительности + долг сна, XP за сон в целевом окне → ось
  «Здоровье». Вход с экрана «Герой». Код: `lib/features/sleep/`.
- **Виджеты главного экрана (Android)**: задачи (до 5), привычки (мини-хитмап + сегодня),
  боссы (HP, до 3); обновление при активности + периодически (~30 мин). Только Android
  (`home_widget`). Код: `lib/features/widgets/`, `android/app/src/main/{kotlin,res}`.

## 2. Креативный backlog (расширение функционала)

### 2.1. 🟡 Фокус и продуктивность
- **«Данж-режим» фокуса** — сессия Помодоро как забег по подземелью; отвлечения = монстры, дошёл до
  конца без выхода из приложения → лут/XP.
- **Заморозка стрика** — предмет «Щит стрика»: спасает серию за пропущенный день.
- **Шаблоны привычек и цепочки** — готовые наборы («Утро чемпиона») в один тап.
- **Эмбиент-музыка/звуки** для фокуса (дождь, таверна, лес).

### 2.2. 🟡 Персонаж и прогрессия
- **Классы персонажа** (Воин/Маг/Мудрец) с бонусами к категориям привычек.
- **Дерево талантов** сверх радара навыков.
- **Рейды** — эпические многошаговые цели с общим HP-баром боссов (развитие текущих боссов-целей).

### 2.3. 🟡 Рефлексия и аналитика
- **Дневник + трекинг настроения**, завязанный на XP; вечерняя рефлексия из блока цитат.
- **Календарь-хитмап** активности (как GitHub-контрибьюшены).
- **Еженедельный/годовой «журнал квестов»** — авто-отчёт с итогами и графиками.

### 2.4. 🟢 Вовлечение и платформа
- **Виджет на домашний экран** — цитата дня + квесты на сегодня.
- **Сезонные ивенты** — лимитированные тематические челленджи и косметика.
- **Быстрое действие/уведомление** для отметки привычки без открытия приложения.

### 2.5. 🟢 Социальное (после облака)
- Друзья, лидерборды, совместные челленджи (уже заложено в архитектуре, см. `ARCHITECTURE_RU.md`).
- Обмен цитатами/карточками достижений.

## 3. Технический долг и известные ограничения

- **Release-подпись.** APK сейчас подписывается debug-ключом. Для Google Play — сгенерировать
  `key.jks`, настроить `signingConfig` (ключ и `key.properties` — вне git).
- **Web + SharedArrayBuffer.** Drift на web использует fallback `sharedIndexedDb` (работает, но
  медленнее). Для полноценного WASM-режима нужны заголовки COOP/COEP (cross-origin isolation) на хостинге.
- **flutter_timezone / KGP.** Плагин применяет старый Kotlin Gradle Plugin — будущие версии Flutter
  могут потребовать миграции; следить за обновлениями плагина.
- **iOS** — отложен по замыслу; при возврате настроить иконки/сплэш/подпись отдельно.
- **Supabase (Фаза 2)** — облако, авторизация, синхронизация ещё не реализованы; см. `ARCHITECTURE_RU.md`.
