# RPG Task Tracker

Мобильное приложение-трекер задач, привычек и целей в **RPG/фэнтези-стиле** с геймификацией.

## Что это

- 📋 **Задачи / привычки / цели** — с категориями, расписаниями и подзадачами.
- 🌹 **Роза навыков** — радарная диаграмма прокачки по осям жизни (Здоровье, Спорт, Учёба, Работа, Творчество, Отношения; оси редактируются).
- ⭐ **Геймификация** — опыт (XP), золото и премиум-кристаллы за выполнение; уровни, стрики, достижения, ежедневные квесты, боссы-цели.
- 🧙 **Персонаж** — герой-аватар с прокачкой и экипировкой + питомец-маскот.
- 🛒 **Кастомизация** — темы, иконки, аватары, звуки за внутреннюю валюту.
- ⏱️ **Помодоро + режим концентрации** — таймер и фокус (мягкий: DND/полноэкранный; жёсткий: блокировка приложений).
- 🔔 **Уведомления поверх экрана** — push, полноэкранные будильники, плавающий оверлей.
- 👥 **Гильдии** — объединяйтесь с друзьями/коллегами по коду приглашения, общий рейтинг участников (см. [GUILDS_SETUP_RU.md](GUILDS_SETUP_RU.md)).
- 🧾 **Журнал наград** — полная история начислений XP/золота/кристаллов со сверкой счётчиков.
- 🗄️ **Архив** — архивирование задач и заметок, поиск и фильтрация по спискам.
- ☁️ **Офлайн-first + облако** — работает без интернета, синхронизируется через Supabase.

## Платформы

Android (основная), Web и Windows Desktop. iOS отложен.

## Технологии

Flutter · Riverpod · Drift (SQLite) · Supabase · go_router · fl_chart · flutter_local_notifications.

## Документация

- [SETUP_RU.md](SETUP_RU.md) — установка окружения разработки (Windows).
- [ARCHITECTURE_RU.md](ARCHITECTURE_RU.md) — архитектура, структура кода, синхронизация.
- [GAMIFICATION_RU.md](GAMIFICATION_RU.md) — формулы опыта, валюты, уровней.
- [ROADMAP_RU.md](ROADMAP_RU.md) — дорожная карта: советы/цитаты, мотивация, backlog идей.

## Быстрый старт (после установки окружения)

```powershell
flutter pub get
dart run build_runner build --delete-conflicting-outputs
flutter run --dart-define=SUPABASE_URL=<url> --dart-define=SUPABASE_ANON_KEY=<key>
```
